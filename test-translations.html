4. Istruzioni Test

📋 Come Testare le Traduzioni:

  1. Apri una pagina admin (Dashboard o Users)
  2. Clicca sul language switcher nell'header (IT Italiano)
  3. Seleziona una lingua diversa (EN, FR, ES, DE)
  4. Verifica che i testi cambino:
    • Titolo della pagina
    • Sottotitolo
    • Pulsanti (Esporta, Aggiorna)
    • Statistiche (Utenti Totali, Adozioni, etc.)
    • Navigazione (Dashboard, Gestione Utenti)
    • User menu (Vai al Sito, Esci)
